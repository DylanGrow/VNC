Architectural Overview

Codebase Directory Structure

backend/
main.py # Primary FastAPI Server & lifespans
auth.py # JWT Token generator
rate_limit.py # Sliding connection limiter
screen_capture.py # Low-latency MSS routines
input_handler.py # Event whitelisting & scaling
clipboard.py # Host OS Clipboard observer
audit.py # Encrypted Security audit-logging
metrics.py # Session latency and performance
frontend/src/
main.ts # Entry initialization bundle
app.ts # Coordinator & polling hooks
renderer.ts # Double-buffered Canvas pool
connection.ts # WebRTC & WebSockets HUD UI
reconnection.ts # Exponential Backoff client
validation.ts # Type-safe Zod schema engines

Python Backend Layer

Runs as a high-performance ASGI service. Leverages MSS API framework, AES-GCM secure storage formats, and asyncio tasks to stream updates and observe clipboard events safely.

TypeScript Client Layer

Wired through Vite compiler bundles. Manages off-screen image pool rendering, dynamic resolution scaling, secure token refresh, and reconnect states safely.

Setup & Execution

Terminal - Setup Backend virtual environment 
# Navigate to backend directory
cd backend

# Create and activate python environment
python -m venv venv
.\venv\Scripts\activate # Windows
# or: source venv/bin/activate (Linux/Mac)

# Install requirements & environment settings
pip install -r requirements.txt
copy .env.example .env

# Run FastAPI reload daemon
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
Terminal - Run Vite Dev bundler 
# Navigate to frontend
cd frontend

# Install project dependencies
npm install

# Start the development client server
npm run dev

Security & Performance Features

JWT Lifecycle & Session Management

Automatic token rotators exchange cryptographic tokens every 15 minutes, blocking invalid sessions and brute-force access attempts.

Boundary & Input Verification

Validates and locks incoming mouse coordinates strictly within [0.0, 1.0]. Whitelists keyboard triggers to prevent terminal escapes or OS execution payloads.

Flicker-free Double Buffering

Utilizes off-screen canvas buffers synced with browser frame updates (`requestAnimationFrame`) to prevent layout reflows or sequence stutter.
